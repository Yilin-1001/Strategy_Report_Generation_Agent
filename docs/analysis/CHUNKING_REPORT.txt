   - 跳过 .pdf, .docx, .doc 文件

   - 自动分类处理

3. Metadata提取 / Metadata Extraction
   - 生成唯一 doc_id
   - 提取文件名作为 title
   - 提取页码信息 (page_number)
   - 提取章节标题 (section_title)
   - 提取日期 (created_at)
   - 提取路径标签 (tags)

   - 向量维度: 1024

6. 向量存储 / Vector Storage
